# 갤러리 커스터마이징 가이드

## 개요

갤러리 페이지는 이제 완전한 커스터마이징 기능을 제공합니다. 갤러리 높이를 동적으로 조정하고, 커스텀 액자와 배경을 직접 만들 수 있습니다.

## 주요 기능

### 1. 갤러리 높이 동적 조정

**목적**: 작품이 많을 때 모든 작품을 배치할 수 있는 충분한 공간 제공

**사용 방법**:
1. 갤러리 페이지의 설정 버튼 (⚙️) 클릭
2. "기본 설정" 탭 선택
3. "갤러리 높이" 슬라이더 조정 (300px ~ 2000px)
4. 변경사항이 즉시 적용됩니다

**권장 높이**:
- 작품 1-3개: 600px (기본값)
- 작품 4-6개: 800-1000px
- 작품 7-10개: 1200-1500px
- 작품 10개 이상: 1500-2000px

### 2. 커스텀 액자 생성

**목적**: 자신의 스타일에 맞는 독특한 액자 디자인 만들기

**사용 방법**:
1. 갤러리 페이지의 설정 버튼 (⚙️) 클릭
2. "액자 생성" 탭 선택
3. 다음 항목들을 설정합니다:

#### 액자 설정 항목

| 항목 | 범위 | 설명 |
|------|------|------|
| **액자 이름** | 텍스트 | 액자를 구분하기 위한 이름 (예: "내 스타일 액자") |
| **테두리 두께** | 1-30px | 액자의 테두리 굵기 |
| **테두리 색상** | 색상 선택 | 테두리의 색상 (색상 팔레트 또는 16진수 코드) |
| **테두리 스타일** | 6가지 | 실선, 점선, 점, 이중선, 릿지, 그루브 |
| **배경 색상** | 색상 선택 | 액자 내부의 배경색 |
| **패딩** | 0-30px | 이미지와 테두리 사이의 거리 |
| **그림자 크기** | 0-20px | 액자의 그림자 크기 (0이면 그림자 없음) |

**미리보기**: 설정을 변경하면 실시간으로 미리보기가 업데이트됩니다

**예시 1: 클래식 우드 액자**
```
이름: 클래식 우드
테두리 두께: 12px
테두리 색상: #8B7355 (갈색)
테두리 스타일: 실선
배경 색상: #F5F1E8 (아이보리)
패딩: 8px
그림자 크기: 8px
```

**예시 2: 미니멀 화이트**
```
이름: 미니멀 화이트
테두리 두께: 4px
테두리 색상: #FFFFFF (흰색)
테두리 스타일: 실선
배경 색상: #FFFFFF (흰색)
패딩: 6px
그림자 크기: 2px
```

**예시 3: 보헤미안 스타일**
```
이름: 보헤미안
테두리 두께: 6px
테두리 색상: #D4AF37 (금색)
테두리 스타일: 점선
배경 색상: #FFF8DC (크림색)
패딩: 12px
그림자 크기: 10px
```

### 3. 커스텀 배경 생성

**목적**: 갤러리의 분위기를 표현하는 배경 이미지 설정

**사용 방법**:
1. 갤러리 페이지의 설정 버튼 (⚙️) 클릭
2. "배경 생성" 탭 선택
3. 배경 이름과 이미지 URL 입력
4. 미리보기 확인 후 "배경 생성" 클릭

#### 배경 설정 항목

| 항목 | 설명 |
|------|------|
| **배경 이름** | 배경을 구분하기 위한 이름 |
| **이미지 URL** | 배경으로 사용할 이미지의 웹 주소 |

**이미지 URL 찾기**:

1. **Unsplash** (무료, 고품질)
   - https://unsplash.com 방문
   - 원하는 이미지 검색
   - 이미지 클릭 → 우클릭 → "이미지 주소 복사"

2. **Pexels** (무료)
   - https://www.pexels.com 방문
   - 이미지 다운로드 또는 URL 복사

3. **Pixabay** (무료)
   - https://pixabay.com 방문
   - 이미지 검색 후 URL 복사

4. **본인의 서버/클라우드**
   - Google Drive, Dropbox, AWS S3 등에 업로드
   - 공개 링크 생성

**권장 배경 이미지**:
- **해상도**: 1920x1080px 이상
- **파일 크기**: 500KB 이하 (로딩 속도)
- **스타일**: 갤러리 분위기에 맞는 텍스처, 패턴, 색상

**배경 예시**:

| 이름 | 설명 | 추천 이미지 |
|------|------|-----------|
| 갤러리 월 | 흰 벽 텍스처 | 밝은 회색 또는 흰색 벽 사진 |
| 수채화 그라데이션 | 부드러운 색감 | 파스텔 톤의 수채화 배경 |
| 목재 바닥 | 따뜻한 분위기 | 밝은 목재 텍스처 |
| 콘크리트 | 모던한 느낌 | 회색 콘크리트 벽 |
| 천 패턴 | 우아한 분위기 | 린넨이나 캔버스 텍스처 |

## 갤러리 배치 팁

### 효과적인 작품 배치

1. **높이 조정 먼저**: 모든 작품을 배치하기 전에 적절한 높이 설정
2. **액자 선택**: 각 작품의 특성에 맞는 액자 선택
3. **배경 설정**: 작품들이 돋보이는 배경 선택
4. **배치**: 마우스로 드래그하여 자유롭게 배치

### 디자인 조화

**색상 조화**:
- 배경이 밝으면 어두운 액자 선택
- 배경이 어두우면 밝은 액자 선택
- 액자 색상이 작품과 너무 겹치지 않도록 주의

**공간 활용**:
- 큰 작품과 작은 작품을 섞어서 배치
- 작품 사이에 적절한 여백 유지
- 한쪽으로 치우치지 않도록 균형 맞추기

**스타일 통일**:
- 같은 스타일의 액자를 여러 개 사용하면 통일감 있음
- 다양한 액자를 섞을 때는 색상 팔레트 통일

## 데이터 저장

### 커스텀 액자/배경 저장

생성한 커스텀 액자와 배경은 자동으로 저장됩니다:

1. **로컬 스토리지**: 브라우저에 임시 저장
2. **portfolio.json**: 데이터 내보내기 시 포함

### 데이터 내보내기

프로필 페이지에서 "데이터 내보내기"를 클릭하면:
- 모든 커스텀 액자 정보 저장
- 모든 커스텀 배경 정보 저장
- 갤러리 배치 정보 저장

### 다른 기기에서 사용

1. 한 기기에서 데이터 내보내기
2. 다운로드된 JSON 파일 저장
3. 다른 기기에서 프로필 페이지 → "데이터 가져오기"
4. 저장한 JSON 파일 선택

## 트러블슈팅

### Q: 배경 이미지가 안 보여요

**A**: 다음을 확인하세요:
- 이미지 URL이 정확한지 확인
- 이미지가 공개 상태인지 확인
- 브라우저 콘솔에서 오류 확인 (F12)

### Q: 액자가 너무 크거나 작아요

**A**: 갤러리에서 작품 추가 시 크기를 조정할 수 있습니다:
- 작품 추가 후 드래그하여 위치 이동
- 작품 제거 후 다시 추가하여 다른 크기로 배치

### Q: 커스텀 액자/배경이 저장되지 않았어요

**A**: 다음 단계를 따르세요:
1. 프로필 페이지에서 "데이터 내보내기" 클릭
2. 다운로드된 JSON 파일 확인
3. 파일에 액자/배경 정보가 있는지 확인
4. 필요시 다시 가져오기

### Q: 갤러리 높이를 변경했는데 작품이 안 보여요

**A**: 다음을 확인하세요:
- 높이가 충분한지 확인 (최소 300px)
- 작품의 위치가 갤러리 범위 내인지 확인
- 작품을 드래그하여 위치 조정
