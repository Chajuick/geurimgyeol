# 일러스트레이터 포트폴리오 관리 앱

일러스트레이터가 자신의 작업 과정을 체계적으로 관리하고, 완성된 작품을 갤러리 형태로 전시할 수 있는 React 기반 웹 애플리케이션입니다.

## 🎨 주요 기능

### 프로젝트 관리
- **7단계 작업 파이프라인**: 러프 → 선화 → 밑색 → 명암 → 묘사 → 보정 → 작품
- **단계별 이미지 관리**: 각 단계마다 이미지 업로드 및 설명 추가
- **유연한 워크플로우**: 필요한 단계만 활성화하고 순서 변경 가능
- **메타데이터 관리**: 제목, 태그, 사용 툴, 주제, 작업 기간 등 기록

### 갤러리 전시
- **드래그 앤 드롭**: 마우스로 작품을 자유롭게 배치
- **다양한 액자 스타일**: 클래식, 모던, 미니멀, 빈티지, 엔티크 등
- **배경 선택**: 여러 갤러리 배경 중 선택 가능
- **실시간 편집**: 작품 위치, 크기, 회전 조절

### 프로필 관리
- **작가 정보**: 이름, 소개, 웹사이트, 프로필 이미지
- **소셜 링크**: Instagram, Twitter, Behance, ArtStation
- **데이터 백업/복원**: JSON 파일로 모든 데이터 내보내기/가져오기

## 💾 데이터 관리

### 파일 기반 데이터 시스템

이 앱은 `public/data/portfolio.json`에서 데이터를 로드합니다.

```json
{
  "projects": [...],
  "frames": [...],
  "backgrounds": [...],
  "gallery": {...},
  "profile": {...}
}
```

### 데이터 흐름

1. **앱 시작**: `public/data/portfolio.json` 로드
2. **사용자 수정**: 로컬 스토리지에 캐시
3. **데이터 내보내기**: 현재 상태를 JSON으로 다운로드
4. **깃허브 업데이트**: JSON 파일을 저장소에 커밋

## 📖 사용 가이드

### 프로젝트 생성

1. "프로젝트" 메뉴 → "새 프로젝트"
2. 기본 정보 입력 (제목, 태그, 도구 등)
3. 각 작업 단계별로 이미지 업로드
4. "프로젝트 저장"

### 갤러리 전시

1. "갤러리" 메뉴 클릭
2. "작품 추가" 버튼
3. 프로젝트 및 액자 스타일 선택
4. 드래그하여 배치

### 데이터 백업

1. "프로필" 메뉴 클릭
2. "데이터 내보내기" 클릭
3. JSON 파일 다운로드

## 🔄 데이터 동기화

### 로컬 스토리지 (임시 캐시)

- 브라우저에 저장
- 페이지 새로고침 시 유지
- 브라우저 캐시 삭제 시 초기화

### 파일 기반 (영구 저장)

- `public/data/portfolio.json`에 저장
- 모든 사용자가 동일한 데이터 로드
- 깃허브에 커밋하여 버전 관리
